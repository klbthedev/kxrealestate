# Real Estate

## Overview
The **Real Estate** module offers a complete property management solution integrated with Odoo. It is designed to help businesses manage property sales, rental contracts, tenant relationships, and financials effectively.

## Features
- 📁 **Properties Hierarchy Management**
- 📝 **Unit Reservations**
- 📜 **Ownership Contract Handling**
- 🏠 **Tenant Management**
- 💰 **Integrated Invoicing & Accounting**
- 🔁 **Property Refund Processing**
- ✉️ **Email Notifications**
- 🌐 **Website Integration**
- 📊 **Comprehensive Reports for Sales & Rentals**

## Functional Highlights
- **Contract Types:**
  - Ownership Contracts
  - Rental Contracts
- **Wizards:**
  - Installment Payments
  - Refunds
  - Rental Renewals
  - Due & Late Payment Analysis
  - Salesperson Sales Summary
- **Reporting:**
  - Reservation Reports
  - Ownership & Rental Contract Reports
  - Quittance Letters
  - Payment Dues by Customer/Unit
  - Occupancy Reports

## Installation
1. Download the module and place it in your Odoo addons directory
2. Update the module list in Odoo
3. Install the module from the Apps menu
4. Configure the module settings according to your business requirements

## Dependencies
- `account`
- `analytic`

## Setup
- Configure properties, units, and installment templates from the **Configuration** menu.
- Use the **Contracts** menu to manage ownership and rental agreements.
- Reports and payment tracking available from **Reporting**.

## Demo Data
To get started, demo data is included and can be loaded during installation.
